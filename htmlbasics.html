HTML Core Concepts

Learn the essential building blocks of HTML including elements, attributes, text formatting, and comments to create well-structured web pages.

HTML Elements & Tags

HTML elements are the building blocks of web pages. An HTML element is defined by a start tag, content, and an end tag.

HTML Code
<tagname>Content goes here...</tagname>

<!-- Example paragraph element -->
<p>This is a paragraph.</p>

<!-- Example link element -->
<a href="https://example.com">Visit Example</a>

<!-- Empty (self-closing) element -->
<img src="image.jpg" alt="Description">
Result

This is a paragraph.

Visit Example

Key Points

  • Elements are defined with tags: <tagname> and </tagname>
  • Most elements have opening and closing tags
  • Some elements are self-closing (void elements) like <img>, <br>, and <input>
  • Elements can be nested inside other elements to create structure
  • HTML documents are made of nested elements forming a tree structure (DOM)

HTML Attributes

Attributes provide additional information about HTML elements. They are always specified in the start tag and usually come in name/value pairs like name="value".

HTML Code
<!-- Attributes provide additional info -->
<img src="photo.jpg" alt="A beautiful landscape" width="500">

<a href="https://example.com" target="_blank">Open in new tab</a>

<div id="main-content" class="container highlighted">
  <p style="color: blue;">Styled paragraph</p>
</div>
Result

Open in new tab

Styled paragraph

Common Attributes

  • id - Specifies a unique id for an element
  • class - Specifies one or more class names for an element (used for CSS and JavaScript)
  • style - Used for inline CSS styling
  • src - Specifies the source file path for elements like <img> and <script>
  • href - Specifies the URL for links (<a>)
  • alt - Alternative text for images
  • title - Provides additional information (tooltip)

Headings & Paragraphs

Headings and paragraphs are fundamental for structuring text content. HTML provides six levels of headings and the paragraph tag for text blocks.

HTML Code
<h1>Main Heading (Level 1)</h1>
<h2>Subheading (Level 2)</h2>
<h3>Sub-Subheading (Level 3)</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

<p>This is a paragraph. Paragraphs are used for blocks of text. They automatically create space before and after themselves.</p>
<p>This is another paragraph. You can have as many paragraphs as you need to structure your content.</p>
Result

Main Heading (Level 1)

Subheading (Level 2)

Sub-Subheading (Level 3)

Heading 4

Heading 5
Heading 6

This is a paragraph. Paragraphs are used for blocks of text. They automatically create space before and after themselves.

This is another paragraph. You can have as many paragraphs as you need to structure your content.

Best Practices

  • Use headings in hierarchical order (h1 → h2 → h3, etc.)
  • Have only one <h1> per page for SEO
  • Use paragraphs for blocks of text instead of multiple line breaks
  • Headings should describe the content that follows
  • Don't use headings just to make text bigger - use CSS for styling

Text Formatting

HTML provides various tags to format text with special meaning. These tags help create emphasis, importance, and other text effects.

Bold Text

<b> - Bold text without extra importance

<strong> - Important text (semantic importance)

Italic Text

<i> - Italic text for alternative voice

<em> - Emphasized text (semantic emphasis)

Underline & Strikethrough

<u> - Underlined text (non-textual annotation)

<del> - Deleted text and <ins> - Inserted text

Highlighting & Small Text

<mark> - Highlighted text

<small> - Smaller text (side comments)

Superscript & Subscript

<sup> - Superscripttext

<sub> - Subscripttext

Quotations

<q> - Short inline quotation

<blockquote> -

Block-level quotation

HTML Code
<p>
  This is <b>bold text</b> and this is <strong>important text</strong>.
  This is <i>italic text</i> and this is <em>emphasized text</em>.
  We have <mark>highlighted text</mark> and <small>small text</small>.
  Chemical formula: H<sub>2</sub>O
  Mathematical power: 10<sup>3</sup> = 1000
  This is <del>deleted text</del> and <ins>inserted text</ins>.
</p>
Result

This is bold text and this is important text.
This is italic text and this is emphasized text.
We have highlighted text and small text.
Chemical formula: H2O
Mathematical power: 103 = 1000
This is deleted text and inserted text.

HTML Comments

Comments are used to leave notes in your HTML code that aren't displayed in the browser. They help document your code and make it more maintainable.

HTML Code
<!-- This is a single-line comment -->

<!--
This is a multi-line comment
It can span several lines
-->

<div class="container">
  <h2>Section Title</h2>
  <!-- TODO: Add content here -->
</div>

<!--
Commenting out code:
<div>
<p>This code is temporarily disabled</p>
</div>
-->
Result

Section Title

Notice that none of the comments appear in the rendered output.

Comments are only visible in the HTML source code.

Why Use Comments?

  • Document your code for future reference
  • Explain complex sections of code
  • Temporarily disable code without deleting it
  • Leave notes for other developers
  • Mark sections of code (TODO, FIXME, etc.)
