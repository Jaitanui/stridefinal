Introduction to HTML

The foundational language of the World Wide Web that structures content for web browsers. Learn how HTML creates the backbone of every website you visit.

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications. It provides the structural foundation for all content on the web.

HTML uses a system of tags and attributes to define elements within a document. These elements form the building blocks of web pages, organizing content into headings, paragraphs, lists, links, images, and more.

How HTML Works

HTML documents are interpreted by web browsers which render the structured content for users. The process involves:

  1. The browser requests an HTML document from a server
  2. The server sends the HTML file to the browser
  3. The browser parses the HTML and builds the Document Object Model (DOM)
  4. The browser renders the visual representation of the DOM
  5. Users interact with the rendered page
Basic HTML Document Structure
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document Title</title>
</head>
<body>
  <!-- Page content goes here -->
  <h1>Hello World!</h1>
  <p>This is my first HTML page.</p>
</body>
</html>

History of HTML

HTML has evolved significantly since its creation in the early days of the web. Here's a brief timeline of major HTML versions:

1989-1991

HTML Created

Tim Berners-Lee invents HTML while working at CERN to share scientific documents.

1993

HTML 1.0

The first official specification with basic text formatting and hyperlinks.

1995

HTML 2.0

Standardized by IETF, adding forms and table support.

1997

HTML 3.2

W3C recommendation with tables, applets, and text flow around images.

1999

HTML 4.01

Major revision with stylesheets, scripting, and accessibility features.

2014

HTML5

Modern standard with semantic elements, multimedia support, and APIs.

Today, HTML5 is the current standard, providing powerful features for building modern web applications while maintaining backward compatibility.

Core HTML Features

HTML provides the essential building blocks for creating web content. Here are its fundamental features:

Text Structure

Organize content with headings, paragraphs, lists, and semantic text tags for meaning and accessibility.

Hyperlinks

Create connections between pages with anchor tags, enabling the "web" of interconnected documents.

Media Elements

Embed images, audio, video, and other media content directly into web pages.

Tables

Display tabular data with rows, columns, headers, and captions.

Forms

Collect user input through various controls like text fields, checkboxes, and buttons.

Semantic Elements

Use meaningful tags like <header>, <article>, and <footer> to improve structure.

HTML in the Modern Web

Today's HTML works alongside CSS and JavaScript to create complete web experiences:

HTML

The structure and content of the page

"What the content is"

CSS

The presentation and styling

"How the content looks"

JavaScript

The behavior and interactivity

"How the content behaves"

HTML + CSS + JavaScript Integration
<!-- HTML Structure -->
<button id="myButton">Click Me!</button>

<!-- CSS Styling -->
<style>
#myButton {
  background: #3b82f6;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 4px;
}
</style>

<!-- JavaScript Behavior -->
<script>
document.getElementById('myButton').addEventListener('click', () => {
  alert('Button clicked!');
});
</script>
